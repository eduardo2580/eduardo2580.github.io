Información del Sitio

Este sitio web de portafolio personal se construye utilizando tecnologías web modernas incluyendo HTML5, CSS3, JavaScript y Three.js para gráficos 3D.

El sitio presenta un diseño orbital 3D interactivo con soporte de fallback elegante para navegadores más antiguos.

Seguridad y Privacidad

Protección de Datos

No se recopilan ni almacenan datos personales en este sitio web. Todas las interacciones son solo en el lado del cliente.

Seguridad del Contenido

Este sitio web utiliza encabezados de Política de Seguridad de Contenido (CSP) estrictos para proteger contra vulnerabilidades web comunes.

HTTPS Solo

Este sitio se sirve solo a través de HTTPS para garantizar una comunicación segura.
