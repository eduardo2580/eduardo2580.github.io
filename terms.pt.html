Informações do Site

Este site de portfólio pessoal é construído usando tecnologias web modernas incluindo HTML5, CSS3, JavaScript e Three.js para gráficos 3D.

O site apresenta um design orbital 3D interativo com suporte de fallback elegante para navegadores mais antigos.
